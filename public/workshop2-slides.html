Getting Started

Create Your Workspace

Before we start, let's set up our project folder:

  1. Open Finder (Mac) or File Explorer (Windows), go to your Desktop, and create a new folder called my-website
  2. In VS Code, go to File → Open Folder and select the my-website folder you just created
  3. Click the Claude icon in the top-right of VS Code to open the chat panel — this is where we'll work for the rest of the workshop

Claude Code works best when it's inside a project folder — this tells it where to save the files it creates for you.

Building Your Brand
Wow Moment #1

The Braindump

This is where the magic starts. Type this into the Claude Code chat panel (or speak it with WisprFlow / Talktastic):

I want to build a website. Interview me — ask me 2-3 questions one at a time, focusing on what would be most useful for creating a brand brief (e.g. who it's for, what makes it different, the vibe I want). Then create a brand brief with a name, tagline, color palette, and visual direction. click to copy

We'll paste every prompt in the chat so you can copy it. Answer Claude's questions honestly — this is the foundation of everything we build tonight.

This is the interview pattern. When you don't know what you want, tell Claude to ask you questions first. The more it understands about your situation, the better its output will be.

Building Your Brand

Add a Design Plugin

Before we build, let's give Claude a design superpower. Type these two commands into the terminal (not the chat panel), one at a time:

/plugin marketplace add nextlevelbuilder/ui-ux-pro-max-skill click to copy

Then install it:

/plugin install ui-ux-pro-max@ui-ux-pro-max-skill click to copy

This plugin teaches Claude professional UI/UX design patterns — color palettes, typography, layouts, and more. It'll make everything we build tonight look significantly better.

Plugins are Claude's app store. They add specialized knowledge and skills. This one turns Claude into a design-savvy builder.

Wow Moment #2

Build Your Website

Now the big one — turn that brand brief into a real website:

Based on the brand brief, build me a complete, beautiful landing page. Include a hero section (the big banner at the top), an about section, what I offer, testimonials, and a call-to-action button. Make it modern, look great on phones, and ready to put on the internet. click to copy

When Claude asks permission to create a file — say yes. It's safe.

To see your website: find index.html in VS Code's left panel. Right-click it → Reveal in Finder (Mac) or Reveal in File Explorer (Windows). Then double-click the file — it opens in your browser.

You just built a complete landing page. If it doesn't look quite right, don't worry — we'll refine it next.

Iterate in Plain English

Your website is live in the browser. Now make it yours — just describe what you want:

Make the top banner more dramatic — bigger headline, add a subtle gradient background, and make the main button pop more. click to copy

Or try your own — here are ideas:

  • "Add an email signup form that looks clean and modern"
  • "Change the color scheme to feel more warm and inviting"
  • "Add a section that shows three things I offer, side by side"
  • "Make it feel more like Apple's website"

Refresh your browser after each change (⌘R Mac · Ctrl+R Windows).

Level Up

Cruise Control

Tired of clicking "Allow" every time? Let's speed things up.

Since we're in our own empty project folder with no sensitive files, we can let Claude work without asking permission each time.

  1. In the Claude Code chat panel, look for the control that says "Ask before edits"
  2. Click it and select Auto-accept

Think of it like telling your employee: "I trust you — just show me the result."

When is this safe? In a dedicated project folder like tonight — with no passwords, personal files, or sensitive data. Turn it off when working with anything sensitive.

Wow Moment #3

Go Live

Time to put your website on the internet. Claude will set up everything you need:

I want to deploy my website to Vercel so I get a live URL I can share. Check if my computer has everything it needs — if anything is missing, install it for me using the fastest method possible. Then deploy using npx vercel and walk me through any steps I need to do. click to copy

Vercel may open in your browser and ask you to log in — use the same email you signed up with at vercel.com.

Notice what just happened. Claude figured out what your computer was missing and installed it for you. You didn't need to know what those tools were — you just said what you wanted and Claude handled the details.

You now have a live URL. Share it in the chat.

Text it to a friend. Post it on social media. It's real. It's yours.

Wow Moment #4

Marketing Strategy

Your site is live. Now let's get people to it. First, make sure web search is enabled — look for the globe icon in the Claude Code panel and make sure it's toggled on. Then paste this:

Search the web for viral marketing patterns and successful launches in my space. Then create a marketing strategy for my website — include 3 ready-to-post social media captions (for Instagram, X/Twitter, and LinkedIn) that I can copy and post right now. Include my live URL. click to copy

Claude is searching the web for real trends in your industry and building a strategy around them.

Web search is a superpower. Claude can find real examples, competitors, viral hooks, and trending formats — then apply them to your specific brand and audience.

Level Up

Power User Techniques

Models — The Thinking Dial

Claude has different "brains" you can switch between. The default works great — switch when you need more speed or more depth.

Haiku

Fastest. Quick, simple tasks. Lightning speed.

Sonnet

Default. Fast and good for most things.

Opus

Slowest but thinks much deeper. Complex, multi-step tasks.

Modes — How Claude Works

You've already used some of these tonight. Here's the full picture:

Plan Mode

Claude creates a plan and asks for your OK before acting. Great for big tasks.

Auto-Accept

What we turned on earlier — Claude works without asking permission each time.

Ask Every Time

The default — Claude asks your permission before every action. Full control.

When Claude Asks Permission

Level What It's Doing Examples
Always safe
Say yes		 Creating, reading, or writing files in your project. Creating your website files, reading your project, writing styles
Glance first
3 seconds		 Running behind-the-scenes commands. Installing tools. Deploying. Installing tools, deploying your website, saving your work
Pause & think
Rare		 Sending emails or messages. Deleting files. Connecting to services with your passwords or login info. Anything that sends data outward

Bottom line: Claude is designed to be safe. You are always in control.

Power user shortcut: If you want to skip all permission prompts permanently, go to Extensions → Claude Code → Settings and enable "Dangerously skip permissions". Only use this in safe project folders — never with sensitive files.
