Context Is Everything

What Is Context?

  • Claude can see your files and folder structure
  • The more relevant info it has, the better it performs
  • This is why you're working in a project folder
  • After a long conversation, start fresh — like giving Claude a clean whiteboard

Keep It Clean

  • Different projects = different conversations
  • Think of it like different notebooks for different subjects
  • Don't mix your Life OS work with a different project

Models & Modes

Models — The Thinking Dial

Haiku

Fastest. For quick, simple tasks. Lightning speed.

Sonnet

Default. Fast and good for most things. What you're using tonight.

Opus

Slowest but thinks much deeper. For complex, multi-step tasks.

Modes — How Claude Works

Plan Mode

Claude creates a plan first and asks for your approval before doing anything. Great for reviewing before Claude acts.

Auto-Accept

Claude goes ahead without asking permission each time. Use when you trust the task and don't want to click Allow 50 times.

Tonight, keep normal mode. Auto-accept is for when you're comfortable.

When Claude Asks Permission

Level What It's Doing Examples
Always safe
Say yes		 Creating, reading, or writing files. Running analysis scripts. Installing common packages. Creating folders, reading Notes, writing HTML, making PDFs
Glance first
3 seconds		 Running terminal commands. Accessing Mac apps. Downloading packages. Terminal commands, Calendar access, installing libraries
Pause & think
Rare		 Sending emails or messages. Deleting files. Connecting to services with your credentials. Anything that sends data outward

Bottom line: Claude is designed to be safe. You are always in control.

Level Up

Create a Skill

A skill is a reusable command you can run anytime:

Create a skill called /weekly-review that does the following when I run it: look through all my notes for anything that's changed since the last time I ran this skill, summarize what I accomplished, list what's still pending, suggest my top 3 priorities for next week, and save the output as a markdown file with today's date. click to copy

Now test it — type /weekly-review

Every Sunday night, one command, and you're organized for the week.
