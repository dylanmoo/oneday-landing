## Workshop Flow

1. Setup check — ensure everyone has VS Code + Claude Code
2. Import Apple Notes into a life-os folder (interview pattern)
3. Analyze themes across imported notes
4. Build a personal HTML dashboard
5. Create a reusable `/morning-briefing` skill
6. Connect Apple Calendar to the skill
